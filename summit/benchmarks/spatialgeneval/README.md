# SpatialGenEval Benchmark Adapter

This module implements the adapter for the SpatialGenEval benchmark (arXiv:2601.20354).

## Overview

SpatialGenEval benchmarks the spatial intelligence of Text-to-Image models using information-dense prompts and multi-choice QA.

## Usage

This adapter allows running the benchmark within the Summit platform.
Requires `SPATIALGENEVAL_ENABLED=true` to run.

## Data

Data is not included in this repository.
