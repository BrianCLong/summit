### 4. Product → Revenue Feedback Loop

**Top 5 Deal-Blocking Gaps Ranked by Revenue Impact:**
1.  **Lack of easy "Bring Your Own Model" (BYOM) configuration (Commercial/Gov):** Security-conscious clients refuse to use external APIs (OpenAI) and want to use internal private models (Llama 3, internal hosted).
2.  **Air-Gapped Deployment Friction (Gov/Defense):** Deployment scripts rely too heavily on internet access (pulling docker images, npm packages at runtime).
3.  **Single Sign-On (SSO) / SAML Integration (Enterprise):** Essential for crossing the enterprise security review finish line.
4.  **Out-of-the-Box Connectors for Legacy Systems (Enterprise):** Need easy connectors for Sharepoint, older SQL databases.
5.  **Granular Export/Reporting (All):** Ability to export the Graph/Timeline views directly into presentation-ready formats (PPTX, PDF) for leadership briefings.

**Smallest Viable Product Changes:**
1.  **BYOM:** Add environment variables/UI config to easily swap the LLM endpoint URL to a generic OpenAI-compatible local server (e.g., vLLM or Ollama).
2.  **Air-Gapped:** Provide a consolidated tarball of all necessary Docker images and an offline Helm chart.
3.  **SSO:** Integrate Passport.js or standard OIDC library specifically for SAML/Azure AD.
4.  **Connectors:** Build a generic JDBC/ODBC connector rather than specific legacy system connectors.
5.  **Export:** Add a simple "Export to PNG" button on the graph/timeline panes.

**Estimated Lift:**
*   Addressing BYOM and Air-Gapped deployments unlocks 80% of the Gov/Defense pipeline.
*   SSO reduces enterprise sales cycle time by 3-4 weeks (bypassing extended security exception reviews).
