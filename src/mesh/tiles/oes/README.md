# Orbital Espionage Shield (OES) & Ephemeral Orbit Twin (EOT)

## Overview
OES/EOT is a Summit-native module designed to build a digital twin of space-to-ground AI infrastructure and defend against orbital espionage campaigns.

## Features
- **Ephemeral Orbit Twin (EOT)**: Graph-based twin of satellites, ground stations, and ephemeral AI workloads.
- **Inference Surface Modeling (ISM)**: Detects and mitigates inference risks from orbital observables.
- **Espionage Game**: Agentic attacker-defender simulation.
- **Pattern Shaping**: Proactive signal shaping to reduce leakage.

## Security
- Default OFF.
- Defensive only.
- Strict data minimization and never-log policies.
