# NarrativeOps IP Plan

## Summit Readiness Assertion
IP handling is aligned with the Summit Readiness Assertion (`docs/SUMMIT_READINESS_ASSERTION.md`) and GA governance requirements.

## Source List & Usage Policy
- **Nature / npj Complexity (2026)**: method reference only; no text or code reuse.
- **Frontiers Human Dynamics (2025)**: method reference only; no text or code reuse.
- **arXiv (2025)**: taxonomy reference only; no text or code reuse.
- **OUP Communication Theory (2025)**: conceptual guidance only; no text or code reuse.
- **DiscoveryAlert (2026)**: secondary source; claims **Deferred pending primary-source verification**.
- **ORF Online (2025)**: access restricted; requirements **Deferred pending primary-source verification**.

## No-Copy Checklist (Required)
- [ ] No verbatim text copied into code or docs.
- [ ] No figures or tables reproduced.
- [ ] Implementations are clean-room and method-level only.
- [ ] All references are citations to sources, not extracted text.
- [ ] Evidence artifacts contain only derived outputs and hashed references.

## Reviewer Sign-Off Requirement
Every PR that references external research must include a reviewer sign-off attesting to clean-room implementation and source compliance.

## Allowed Outputs
- High-level method summaries.
- Taxonomy mapping aligned to public descriptions.
- Deterministic evaluation metrics without reproducing copyrighted content.
