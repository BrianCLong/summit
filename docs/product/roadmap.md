# Summit Product Roadmap

## Near-Term Features
* **Proof Moat MVP**: Deliver an irreducible enterprise-grade stack showcasing deterministic, verifiable chain-of-custody. Essential for initial Go-To-Market wedge targeting Tier-1 Financial Services.
* **Brand Takedown Use Case Demo Path**: Establish a robust, repeatable demonstration workflow identifying typosquat domains, automatically capturing required evidence (screenshots, DNS/cert data), and exporting legally admissible signed evidence bundles.
* **MCP Integration Stabilization**: Standardize the Model Context Protocol (MCP) as the primary tool integration substrate, ensuring sandboxed, governed access for agent tool connectivity.
* **Core Governance Enforcement Policies**: Codify critical platform boundaries (e.g., API request limits, restrictive data scraping rules) explicitly as code to guarantee reliable, auditable playbook execution.
* **Vulnerability Prioritization Framework**: Develop deterministic scoring for identified vulnerabilities relying solely on explicit, explainable lifecycle stages and supporting evidence, bypassing opaque scoring heuristics.

## Mid-Term Features
* **Advanced GraphRAG Multi-Hop Capabilities**: Enhance Neo4j traversal strategies to efficiently manage complex, deep-link querying to synthesize insights from highly interconnected, massive threat intelligence graphs.
* **Integrated Evidence Ledger Enhancements**: Refine and harden the automatic compilation and cryptographically secure stamping of evidence bundles (`report.json`, `stamp.json`), ensuring they natively support seamless ingestion into customer legal workflows.
* **Premium Third-Party Intelligence Connectors**: Expand the external API ecosystem with structured, policy-aware adapters for premium data providers (e.g., Social Links, Orpheus), relying strictly on a BYOK (Bring Your Own Key) operational model.
* **Optional Copilot SDK Adapter**: Introduce a safe, strictly constrained SDK adapter aimed at customers already standardizing their tooling on Copilot, without compromising the platform’s foundational policy plane.

## Long-Term Features
* **Deterministic Replay & Full Audit Automation**: Realize perfect byte-for-byte reproducibility of complex, multi-agent investigations, offering unparalleled transparency and compliance-grade proof of exactly how conclusions were drawn.
* **Automated Redaction & Compliance Enforcement**: Develop intelligent workflows to automatically redact Personally Identifiable Information (PII) or classified data before evidence export, guided by the central policy engine.
* **Dynamic, Context-Aware Investigation Synthesis**: Transition from purely predefined playbooks to advanced, context-aware autonomous agent orchestration while preserving deterministic verification and rigorous evidentiary standards.
* **Expanded Enterprise Deployment Models**: Support highly customized, isolated deployment patterns (On-Premises, VPC) optimized for government, defense, and top-tier enterprise environments requiring extreme data sovereignty.
