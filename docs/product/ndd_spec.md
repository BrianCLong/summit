# NDD Product Spec (MVP PR1–PR3)

## Analyst User Stories
- As an analyst, I need early weak-signal watchlists to detect narrative seeding.
- As an analyst, I need handoff alerts to track fringe → semi-legit transitions.
- As an analyst, I need robustness dashboards to see narrative persistence after corrections.
- As an analyst, I need comparative framing views across topics.

## MVP Scope (PR1–PR3)
- Narrative cluster list with evidence IDs.
- Origin density and persistence dashboards.
- Handoff alert feed with explainability payload.

## Explainability Requirements
- Top contributing features per alert.
- Exemplar posts with tier timelines.
- Evidence bundle links for each alert.

## Governance UX Requirements
- Approval flow for exports and privileged unmasking.
- Audit views for evidence ID trails.
- Policy status banner on every alert.

## Success Metrics
- Precision/recall on labeled fixtures.
- Handoff Lead-Time (HLT) vs virality baseline.
- Analyst time-to-triage.
- False positive rate by domain.
