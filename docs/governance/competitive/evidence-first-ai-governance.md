# Evidence-First AI Governance — Competitive Positioning

## Positioning Summary

Summit ships **traceability primitives** that make audit readiness a normal output of runtime operation: ledger + deterministic audit bundles + continuous governance signals.

## Competitive Split

- **Palantir-like**: audit trails by default → Summit delivers ledger + exportable audit bundles.
- **Microsoft-like**: ambient compliance → Summit delivers continuous scorecards + remediation queue.
- **Dataiku-like**: evidence automation → Summit delivers event-tied artifacts and deterministic exports.

## Claim Boundaries

- No certification claims.
- No legal compliance guarantees.
- Claim: traceability primitives + audit bundle export + continuous governance signals.
