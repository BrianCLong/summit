# Recommendations Policy

> Owner: @summit/governance
> Last-Reviewed: 2026-02-25
> Evidence-IDs: EVD-PLACEHOLDER
> Status: active


Only the following next steps are allowed to be recommended by the system:

- monitor
- collect_more_context
- escalate_to_comms
- escalate_to_security
- request_fact_check
- prepare_executive_briefing
- report_to_platform
- engage_legal_review

Any other recommendation (especially offensive or manipulative ones) is strictly prohibited.
