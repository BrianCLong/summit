# Core Service Dashboards

This directory contains tool-agnostic specifications for the "Golden Signals" dashboards required for all core services.

## Specifications

- [Latency](./latency.md) - Request duration and distribution.
- [Errors](./errors.md) - Error rates, codes, and impact.
- [Throughput](./throughput.md) - Request volume and concurrency.
- [Saturation](./saturation.md) - Resource utilization and limits.
- [Business KPIs](./business-kpis.md) - Domain-specific key performance indicators.
