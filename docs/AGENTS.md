# Summit Agents

**Scope:** The cognitive layer where active work happens.
**Focus:** Orchestration, task autonomy, and human-in-the-loop workflows.

## Capabilities
- **Multi-Agent Orchestration:** `Maestro` engine for coordinating specialized agents.
- **Cognitive Skills:** Tool use, reasoning, and context management.
- **Active Investigations:** Agents that proactively explore the graph to answer questions.

## Key Components
- **Maestro:** The central orchestration service.
- **Skills Pack:** Reusable cognitive skills (e.g., OSINT, Financial Analysis).
- **Agent Protocol:** Standard interface for agent-to-agent communication.
