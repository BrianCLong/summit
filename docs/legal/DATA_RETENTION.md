# Data Retention

**Last Updated:** October 2025

## Default Policy

By default, Summit (IntelGraph) adheres to a **"No Retention by Default"** policy for processed intelligence data, except where explicitly configured by the customer.

| Data Category | Retention Period | Default Setting |
| :--- | :--- | :--- |
| **Customer Intelligence Data** | Configurable | **None (Ephemeral)** |
| **LLM Inputs/Outputs** | Configurable | **None (Ephemeral)** |
| **System Logs** | 30 Days | **30 Days** |
| **Audit Trails** | 1 Year | **1 Year** |
| **User Accounts** | Life of Account | **Indefinite** |

## Configuration

Retention policies for Intelligence Data and LLM interactions can be configured in the System Settings.

*See also [Data Retention Policy](./DATA_RETENTION_POLICY.md) for detailed principles.*
