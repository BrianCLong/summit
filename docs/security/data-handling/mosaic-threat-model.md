# Data Handling: MOSAIC-Style Threat Model

## Classification
- Case metadata: restricted.
- Narrative details: restricted.
- Raw contact content: highly restricted.
- Derived scores and rationale: internal restricted.

## Never-log fields
street_address, phone_number, personal_email, private_message_contents, minor_identifier,
weapon_access_instructions, medical_claim, protected_class.

## Retention
- Raw case payload: 30 days unless legal hold.
- Derived assessment artifacts: 180 days.
- Aggregated non-identifying metrics: 1 year.
