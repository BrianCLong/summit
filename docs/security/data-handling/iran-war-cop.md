# Iran War COP Data Handling

## Classification
- Public-source descriptive intelligence
- Derived analytic judgments
- Operational metadata

## Retention
- Raw records: short retention
- Normalized evidence: versioned retention
- Generated briefs: retained with provenance index

## Never-log list
- precise civilian home/work geolocation
- private contact data
- deanonymizing join keys
- hidden platform tokens/cookies
- model prompts containing user secrets
- any generated target package fields
