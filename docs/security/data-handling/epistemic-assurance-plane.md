# Epistemic Assurance Plane Data Handling

## Classification
* **Public:** URLs, source domain metadata, schema IDs
* **Internal:** support scores, policy decisions, conflict metrics
* **Sensitive:** raw snippets, analyst notes, source handles where reputationally risky
* **Never log:** full raw source text, analyst override freeform notes, private tokens, credential-bearing URLs, user-entered PII

## Retention
Deterministic artifacts retained; raw source payload retention policy must align with existing Summit data handling controls.
