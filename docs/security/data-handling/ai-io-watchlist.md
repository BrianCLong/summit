# Data Classification / Retention / Never-Log for AI-IO Watchlist

## Data Classes
- **Source Citations**: Public
- **Repo Metadata**: Internal Engineering
- **Analyst Notes**: Restricted
- **Platform Account IDs in future live expansion**: Sensitive

## Never Log
- API tokens
- Unredacted user handles from restricted datasets
- IPs / device fingerprints unless explicitly approved
- Moderation decision rationale from external platforms unless public
