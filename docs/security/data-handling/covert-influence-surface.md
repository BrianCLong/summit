# Data Handling: Covert Influence Surface

## Classification
* **Never Log**: Raw internal comms content (default), private audio/video, PII beyond allowlist, tokens.
* **Retain**: Hashes, manifests, derived scores, minimal excerpts (if permitted).

## Residency
* Artifacts must be tagged by region/policy.

## Privacy
* Internal comms ingestion defaults to metadata-only.
* Content ingestion requires explicit opt-in + redaction.
