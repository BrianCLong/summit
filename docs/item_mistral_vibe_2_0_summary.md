# Item Snapshot: Mistral Vibe 2.0

* **ITEM**: “A European AI challenger goes after GitHub Copilot: Mistral launches Vibe 2.0” (VentureBeat, Jan 27, 2026)
* **Primary claim**: Mistral Vibe 2.0 (terminal-based coding agent) + Devstral 2 model family aims at enterprise coding workflows with customization/controls (subagents, clarifications, modes, skills). ([Venturebeat][1])

## Public Feature Details
* **Vibe 2.0**: Custom subagents, multi-choice clarifications, slash-command skills, unified agent modes, automatic updates. ([Venturebeat][1])
* **Devstral 2**: 123B dense, 256K context; Devstral Small 2: 24B; SWE-bench Verified scores (72.2% / 68.0% per Mistral). ([Venturebeat][1])
* **CLI features**: Project-aware context, smart references (@ files, ! shell, slash commands), multi-file orchestration, permissions, config.toml, programmatic mode, “auto-approval” toggle. ([Mistral AI][2])
* **Commercialization**: Bundled with Le Chat plans; Pro $14.99/mo, Team $24.99/seat/mo; Devstral 2 API pricing listed. ([Venturebeat][1])
