## SLAs
- PR review for docs: < 2 business days
- Docs update on release: by code-freeze + 24h

## SLOs
- Broken links < 0.5%
- Stale pages (< lastUpdated 120d) < 5%
- A11y violations = 0 on key pages

## Process
- Weekly stale sweep triage
- Release docs freeze checklist
