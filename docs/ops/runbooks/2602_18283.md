# Runbook: 2602.18283 Evaluation

## Description
This runbook covers the operational tasks to run the HyTRec evaluator locally and understand CI pipeline results.

## Setup
By default, the evaluator feature flag is OFF.

Enable it locally:
```bash
export HF2602_18283_ENABLED=1
```

## Running the Baseline
```bash
python scripts/run_2602_18283.py --config configs/hf_2602_18283.yaml
```

## Interpreting Failures
If the CI job `2602_18283 Reproducibility Gate` fails, check `metrics.json`. The candidate's `delta` improvement relative to the baseline must exceed the `threshold_delta` defined in the config.
