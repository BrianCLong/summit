# Runbook: Cognitive Battlespace Map

## Overview
CBM builds narrative and influence graphs.

## Alert Responses
- `CBM_NARRATIVE_SURGE`: Investigate recent clusters for inorganic amplification.
- `CBM_COORDINATION_CLUSTER`: Review identified actors and asset nodes.
- `CBM_AI_LAUNDERING_RISK`: Assess model outputs mimicking detected narratives.
- `CBM_DATA_VOID_SPIKE`: Add authoritative sources for target locale.
- `CBM_DRIFT_DETECTED`: Check drift report for anomalies.

## Ops Commands
- Deterministic CI mode: `python3 -m summit.cbm --mode replay`
