# Runbook: Covert Influence Surface

## Incident Mode
1. **Freeze CIE**: Stop new simulation runs.
2. **Switch to Fast-Path**: Enable real-time rescoring for specific entities.
3. **Generate Bundle**: Create sealed evidence bundle for legal/comms.

## On-Call Toggles
* Disable external collectors (TruthScan/Blackbird).
* Disable content ingestion.
* Run metadata-only mode.

## Alerts
* Spike in low-integrity artifacts mentioning execs/brands.
* New coordinated cluster emergence.
