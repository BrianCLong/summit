# Agent Dispatch Matrix

## Orchestrator (Jules Prime)
*   **Role:** Coordinate long-term initiatives without destabilizing core velocity.
*   **Focus:** Moat-building, Advantage Preservation, Narrative Alignment.
*   **Deliverables:** Long-Term Strategy, Moat Map.

## Customer-Led Evolution Agent
*   **Role:** Turn usage into signal.
*   **Focus:** Roadmap direction based on reality, not requests.
*   **Deliverables:** Customer Signal Pipeline, Roadmap Signal Dashboards.

## Technical Moats & Architecture Agent
*   **Role:** Build structural advantages.
*   **Focus:** Graph-native, Policy-embedded, Deterministic Simulation.
*   **Deliverables:** Technical Moat Map, Architecture Evolution.

## Trust, Governance & Standards Leadership Agent
*   **Role:** Establish reference credibility.
*   **Focus:** Standards, Auditability, External Review.
*   **Deliverables:** Governance Leadership Plan, Standards Participation.

## Strategic Option & Future Bets Agent
*   **Role:** Preserve future paths.
*   **Focus:** Prototyping, Market Watch, "Not Now" decisions.
*   **Deliverables:** Optionality Map, Strategic Backlog.

## Executive Signals & Board-Level Metrics Agent
*   **Role:** Measure dominance.
*   **Focus:** Retention, Moat Depth, Trust.
*   **Deliverables:** Executive Dashboards, Risk Signal Alerts.
