## Epic 8 — Efficiency & Cost Discipline (make margin a product feature)

1. Create cost per core action metric and assign owners.
2. Implement infra cost budgets per team with alerts.
3. Consolidate vendors and tools; cancel low-usage spend.
4. Optimize data storage/compute via retention tiers and query tuning.
5. Reduce support toil via self-serve, better errors, and automation.
6. Implement pricing/packaging aligned to value and cost drivers.
7. Reduce CI time and incident rate (engineering efficiency).
8. Run quarterly “efficiency sprints” with measured savings shipped.
9. Tie savings to reinvestment into reliability/security debt burn.
10. Track burn multiple and explain changes with honest drivers.
11. Delete projects that consume cost without moving outcomes.
