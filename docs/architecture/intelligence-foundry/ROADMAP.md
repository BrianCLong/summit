# Intelligence Foundry Roadmap (Phased)

## Phase 0: Documentation and Contracts

- Publish spec + data model + API contracts
- Establish evidence bundle schema
- Define agent/tool contract formats

## Phase 1: Governance Substrate MVP

- Policy gateway enforcing asset/model/tool access
- Execution graph capture + deterministic serialization
- Evidence bundle sealing + signature verification

## Phase 2: Foundry Implementations

- Creative Foundry: asset-bound generation + watermarking + provenance
- Analysis Foundry: retrieval provenance + source catalogs + citations

## Phase 3: Compliance and Enterprise

- Controls mapping reports (SOC2/ISO/NIST)
- Quorum approvals and policy-driven workflows
- Drift detection and continuous evaluation

## Phase 4: Ecosystem and Moat

- Multi-foundry orchestration across domains
- Cross-foundry governance with explicit grants
- Marketplace of policy-verified agents/tools (tenant-controlled)
