# INFOWAR Narrative Graph Architecture

## Ontology
- **Nodes:** Narrative, Claim, Actor, Platform, Event, Artifact, Regulation
- **Edges:** AMPLIFIES, REFERENCES, TARGETS, COUPLED_WITH, EVIDENCED_BY

## Propagation Model
Narrative graphs track the spread of claims across platforms and actors, calculating impact scores based on trust-network propagation.

## Evidence Linkage
Every node and edge in the narrative graph must be traceable back to an Evidence Bundle via EVIDENCED_BY relationships.
