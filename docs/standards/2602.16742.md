## Ground Truth Capture

Claims below are grounded in the paper source (`main.tex`) from arXiv.

- `ITEM:CLAIM-01`
  - DeepVision-103K is positioned as a multimodal RLVR dataset with visual diversity, broad K12 coverage, and verifiable answers.
- `ITEM:METHOD-02`
  - Data curation is a 3-stage pipeline: validity filtering, pass-rate difficulty filtering, and query correctness verification.
- `ITEM:RESULT-03`
  - Reported table results include:
    - `Qwen3-VL-8B-DeepVision`: WeMath `85.11`, MMMU_val `71.33`, M3CoT `71.61`.
    - `MiMo-VL-7B-DeepVision`: LogicVista `65.62`, MMMU_val `71.00`, MMMU_Pro `69.19`.
  - Reported gain range versus base Instruct/SFT baselines: `2.91%` to `8.56%`.
- `ITEM:LIMIT-04`
  - Stated limits: imbalanced visual distribution, dependence on strong external verifier models, K12 + unique-answer scope excludes open-ended proof tasks.
- `ITEM:DATA-05`
  - Composition and filtering path includes 3.3M initial pool, 880K after validity filtering, 99K after pass-rate filtering, 77K verified multimodal math, plus 26K visual-logic data.
  - Source license table includes mixed permissions (Apache-2.0, CC BY-NC 4.0, MIT, and Unset entries).
