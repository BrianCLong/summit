# Agentic Context Layer Standards (AEIP)

## Positioning
Summit provides a verifiable governance substrate beneath AI agents orchestrating work across systems.

## Interop Mapping

| Import | Export |
|---|---|
| CRM entity | AEIP validated record |
| ERP invoice | Reconciled entity hash |
| Agent intent | Permission-verified action |

## Non-goals
- Not replacing MDM
- Not building LLM runtime
- Not replacing ERP/CRM
