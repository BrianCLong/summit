# Decentralized AI Standards Mapping

## Purpose

Define machine-verifiable openness and composability checks for decentralized AI claims.

## Standards

| Standard | Role |
| --- | --- |
| OpenAPI | API transparency and compatibility checks |
| SPDX | Model/license disclosure validation |
| W3C DID | Optional decentralized identity support |
| OCI artifacts | Portable model packaging metadata |
| ONNX | Model portability signal |

## Non-goals

- Token issuance design.
- On-chain governance execution.
- Node hosting or infrastructure operations.
