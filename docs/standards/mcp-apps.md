# MCP Apps Standard

## Overview
MCP Apps enable interactive UI capabilities within MCP clients, governed by Summit's policy and evidence framework.

## Integration Pattern
- **Policy**: Deny-by-default, require approval for high-risk actions.
- **Evidence**: All actions produce deterministic evidence bundles (report, metrics, stamp).
- **Verification**: CI verifier enforces manifest, schemas, and policy fixtures.

## Compatibility
- Adheres to Summit Evidence Contract Standard (ECS).
- Future adapters will support Slack, Figma, Canva.
