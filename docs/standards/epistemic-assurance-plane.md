# Epistemic Assurance Plane Standards

## Overview
The Epistemic Assurance Plane decides whether a claim is publishable, actionable, or only hypothesis-grade based on evidence, provenance, uncertainty, and policy.

## Entities
*   `EpistemicClaim`
*   `Evidence`
*   `ProvenanceStep`
*   `EpistemicPolicy`
*   `EpistemicDecision`

## Artifacts
*   `report.json`
*   `metrics.json`
*   `stamp.json`

## Evidence ID Pattern
`EPI-<domain>-<yyyyMMdd>-<slug>-<nnn>`
Example: `EPI-osint-20260307-domain-x-001`
