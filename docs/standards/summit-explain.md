# Summit Explain Standards

**Import/Export Matrix:**
* `import`: local source files
* `import`: repo metadata
* `import`: CI environment variables
* `export`: report.json
* `export`: metrics.json
* `export`: stamp.json
* `export`: optional Postgres rows
* `export`: optional embeddings rows

**Non-goals:**
* not a code generator
* not a full code review replacement
* not a general-purpose IDE
* not a full-blown architecture diagramming suite in v1
