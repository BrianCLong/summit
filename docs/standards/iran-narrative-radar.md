# Standard: Iran Narrative Radar

## Core Concepts
- **Provenance-First:** Every claim must tie to an `evidenceId`.
- **Deterministic:** Analyst replay on golden data must yield stable metrics.
- **Visual Risk:** Attach visual-reuse heuristics (not definitive proof).

## Exclusions
- Not a truth oracle.
- Not a complete computer-vision lab.
- Not a general media monitoring suite.
