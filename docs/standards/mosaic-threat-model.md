# MOSAIC-Style Threat Model Standard (Clean-Room)

This standard defines Summit's clean-room behavioral risk triage pattern: transparent indicators, deterministic scoring, evidence-linked outputs, and graph projection. It is MOSAIC-style and not a proprietary MOSAIC reproduction.

## Non-goals
- No proprietary questionnaire text, tuned weights, or report templates.
- No prediction claim of future violence probability.
- No protected-attribute scoring.
