# Composer 1.5 Subsumption Standard

## Import/Export Matrix

* **Imports:**
    * Task spec (JSON/YAML)
    * Repo context snapshot (paths + hashes)
    * Optional tool manifest
* **Exports:**
    * Evidence bundle (report/metrics/stamp)
    * Summaries
    * Benchmark results
    * Drift report

## Non-goals

* Training RL.
* "Thinking tokens" as model-internal tokens.
* Reproducing Cursor benchmarks exactly.
* Copying UX.
