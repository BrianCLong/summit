# IntelGraph Stakeholder Map

This document identifies the key stakeholders and decision owners for the IntelGraph platform.

| Persona | Role | Primary Responsibility |
|---------|------|------------------------|
| **Product Owner** | Decision Owner | Backlog prioritization, scope alignment, and success metrics. |
| **Architect** | Design Owner | System architecture, ADRs, and technical standards. |
| **Data Engineer** | Implementation | Data modeling, ontology mapping, and graph schema. |
| **API Developer** | Implementation | GraphQL gateway, API contracts, and auth flows. |
| **Ingest Engineer** | Implementation | Batch and streaming pipelines, connector SDK. |
| **Security Officer** | Governance | Data classification, encryption, and threat modeling. |
| **Provenance Lead** | Compliance | Auditability, ledger design, and chain-of-custody. |
| **Frontend Dev** | Implementation | Graph explorer, UX/UI, and accessibility. |
| **AI/ML Scientist** | Implementation | RAG pipelines, vector store, and eval harness. |
| **DevOps Engineer** | Automation | CI/CD, IaC, and environment management. |
| **SRE** | Reliability | Observability, SLIs/SLOs, and chaos engineering. |
| **Compliance Officer**| Governance | Privacy audits, regulatory alignment (PII, retention). |
| **Cost Manager** | FinOps | Unit economics, budget caps, and cost metering. |
| **QA Engineer** | Verification | Acceptance criteria, E2E tests, and quality gates. |
