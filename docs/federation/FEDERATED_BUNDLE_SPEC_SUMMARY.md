# Federated Bundle Spec Summary

This document provides a summary of the federated bundle spec.

## Required Fields

-   `source_repo`
-   `source_sha`
-   `tag`
-   `build_timestamp`
-   `bundle_manifest`
-   `checksums`
-   `artifact_index`

## Artifact Classes

-   `security`
-   `ops`
-   `compliance`
-   `gtm`
-   `cs`
