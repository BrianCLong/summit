## Responsive Behavior

### Desktop (> 1200px)
- Full layout as shown above
- Side-by-side panels
- Expanded navigation

### Tablet (768px - 1200px)
- Stacked layout for main sections
- Collapsible sidebars
- Touch-friendly targets (44px minimum)

### Mobile (< 768px)
- Single column layout
- Bottom navigation bar
- Slide-out panels instead of sidebars
- Simplified event cards

---

## Accessibility Requirements

1. **Keyboard Navigation**: All actions accessible via keyboard
2. **Screen Reader Support**: ARIA labels on all interactive elements
3. **Color Contrast**: WCAG 2.1 AA compliance
4. **Focus Indicators**: Visible focus states
5. **Motion**: Respect `prefers-reduced-motion`
6. **Text Scaling**: Support up to 200% zoom
