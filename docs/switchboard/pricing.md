# Switchboard Pricing Model

**Price on predictable value + governed autonomy, not raw tokens.**

## Core Tiers

### 1) Community (Free)
**$0**
- 1 agent
- **10 runs/day**
- 7-day receipts retention
- Observe + Assist modes (no Autopilot)
- Verified skills only (curated store)
- Cost guardrails on by default (daily cap)

### 3) Power (Builders / Analysts / Heavy Users)
**$39/mo (monthly) / $390/yr**
- Up to 3 agents
- **300 runs/day**
- **365-day receipts retention**
- Advanced tools: browser automation (governed), long-doc ingest, scheduled jobs
- Skill builder: create private skills (still signed + sandboxed locally)
- Higher caps on "expensive actions" with visible budgets

### 4) White-Label Starter (Creator/SMB Brand)
**$99/mo**
- 1 agent-seat (serves up to 5 humans)
- **500 runs/day**
- 180-day retention + exports
- Tenant branding + custom domain
- Org policy overlay + basic RBAC

### 5) White-Label Team
**$299/mo**
- 3 agent-seats (up to 15 humans)
- **2,000 runs/day**
- 365-day retention
- SSO-lite / invite controls
- Compliance exports (bundle ZIP)

## Overage / Add-ons
- **Extra Runs Pack:** $10 = +1,000 runs (valid 30 days)
- **Retention Extension:** $5/mo per additional 90 days (per tenant)
- **Storage Pack:** $10/mo per +50 GB artifacts
- **Premium Models Add-On:** $10/mo unlocks higher-cost model routing with required budgets + alerts

## Value Levers
1. **Retention length** (trust + governance)
2. **Autopilot eligibility** (and what workflows can be delegated)
3. **Verified skills catalog depth**
4. **Agent count / profiles**
5. **Exports** (evidence bundle = enterprise bridge)
6. **Risk controls** (egress allowlists, approvals, kill switch)
