## Summit Global Innovation Fund

Build net-new ecosystem capabilities with structured grant support for plugin builders, integration partners, and applied research teams.

- **Registry:** `.repoos/fund/innovation-fund-registry.json`
- **Grant categories:** `.repoos/fund/grant-categories.json`
- **Proposal intake:** `.repoos/fund/proposals.json`
- **Evaluation output:** `.repoos/fund/evaluation-results.json`

### How to participate

1. Draft a proposal with `proposal_id`, project description, requested funding, and ecosystem impact.
2. Submit through the Innovation Fund CLI (`node scripts/repoos/innovation-fund.mjs fund submit-proposal ...`).
3. Review deterministic evaluation results and approval status through the fund registry.
