# Summit Operating Pack v1

This directory contains the core documents for the Summit Operating Pack.

## Documents

* [**Charter**](./CHARTER.md): Defines the objective, scope, ownership, and cadence.
* [**Milestone Plan**](./MILESTONE_PLAN.md): Tracks the 7 key milestones from kickoff to postmortem.
* [**Risk Register**](./RISK_REGISTER.md): Tracks critical risks and their mitigations.
* [**Decision Log**](./DECISION_LOG.md): Records all key decisions. **If it isn't here, it didn't happen.**
* [**Workstream Tasking**](./WORKSTREAM_TASKING.md): Initial kickoff message template.
* [**Checkpoint Agenda**](./CHECKPOINT_AGENDA.md): Standard agenda for the daily/weekly checkpoint.
