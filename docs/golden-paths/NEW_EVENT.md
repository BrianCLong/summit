# Golden Path: New Event

## Checklist

- [ ] **Definition**: Event type defined in `server/src/provenance/types.ts` or `server/src/events/types.ts`.
- [ ] **Schema**: Event structure follows the Canonical Event Taxonomy.
- [ ] **Context**: Includes `tenantId`, `userId`, `timestamp`, and `traceId`.
- [ ] **Registry**: Registered in the schema registry (if applicable).
