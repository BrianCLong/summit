# Prompt Packets Repository

This directory contains structured prompt templates (Prompt Packets) for various engineering tasks.

## Templates
- [Bugfix](../../tools/promptpacks/templates/bugfix.json)
- [Small Refactor](../../tools/promptpacks/templates/refactor-small.json)
- [Test Addition](../../tools/promptpacks/templates/test-addition.json)

## How to use
1. Select a template.
2. Fill in the Goal, Inputs, Outputs, Constraints, and Non-goals as per [Policy](../../policy/ai-coding/prompt-packet.md).
3. Provide the filled packet to your AI assistant.
4. Verify the output against the Acceptance Tests.
