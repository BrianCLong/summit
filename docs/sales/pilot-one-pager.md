# Standard Paid Pilot (6–8 Weeks)

## Offer
A fixed-scope, paid pilot that turns three mission-critical workflows live with governed evidence, security approval, and a clear expansion path.

## Success Criteria (Hard Gates)
1. **Three workflows live** with measurable outcomes and agreed SLOs.
2. **SLOs met** for latency, availability, and throughput across pilot lanes.
3. **Audit trail & provenance verified** end-to-end.
4. **Security review passed** with documented evidence bundle.
5. **One integration connector shipped/validated** against a production-relevant system.

## Deliverables
- **Evidence Pack** (audit trails, provenance proofs, policy enforcement logs, test results).
- **Expansion quote** with Year 1–3 ramp and package gates.
- **Pilot recap** mapped to mission outcomes and operational metrics.

## Timeline (6–8 Weeks)
- **Week 1–2:** Alignment + access, workflow selection, policy scoping, readiness checks.
- **Week 3–5:** Build + integrate workflows, run quality gates, begin evidence collection.
- **Week 6–8:** Security review, SLO verification, evidence pack delivery, expansion plan.

## What We Need From You
- Named business owner + security lead.
- Access to target systems and data handling constraints.
- Approval for pilot scope and success criteria.

## Commercials
- **Pricing:** Deferred pending pricing gate approval; fixed-fee pilot with defined scope.
- **Conversion Path:** Pilot converts into Team/Business/Enterprise package based on verified gates.
