# Pilot Offer: IntelGraph "Proven Value" Program

**Duration:** 6-8 Weeks (Two Sprints)
**Cost:** [Fixed Price / TBD]

**Goal:** Demonstrate quantifiable value (Time-to-Insight, Audit Completeness) in a customer's specific environment.

## Scope of Work (SOW)

### Phase 1: Deployment & Configuration (Week 1-2)

- **Infrastructure:** Deploy IntelGraph (On-Prem, Private Cloud, or SaaS Isolated).
- **Connectivity:** Configure up to 3 Data Connectors (e.g., SQL, CSV, API).
- **Security:** Integrate with Customer IdP (SSO) and apply OPA Policies.

### Phase 2: Workflow Validation (Week 3-6)

- **Use Case:** Execute 2 primary workflows (e.g., "Insider Threat Detection", "Supply Chain Mapping").
- **Training:** 2x Workshop sessions for Analysts and Admins.
- **Tuning:** Adjust Entity Resolution and Risk Scoring models.

### Phase 3: Review & Expansion (Week 7-8)

- **Evidence:** Deliver "Audit Bundle" showing full lineage of investigations.
- **Metrics:** Report on "Time Saved" vs legacy process.
- **Roadmap:** Plan for Production rollout (HA, DR, Scale).

## Success Criteria

1.  **Analyst Velocity:** Reduce investigation time by >30%.
2.  **Audit Readiness:** Pass a mock compliance audit using generated logs.
3.  **Cost Control:** Operate within defined token/compute budget.

## Deliverables

- Configured Production-Ready Environment.
- Trained Core User Team (approx 5 users).
- Evidence Bundle (System Verification).
- Expansion Plan & ROI Report.
