# Integrity Feature Flags

| Flag | Description | Default |
| --- | --- | --- |
| `INTEGRITY_HARASS_DETECTOR_ENABLED` | Enables the harassment burst detector. | `false` |
| `INTEGRITY_OBSERVATORY_EXPORT_ENABLED` | Enables standardized observatory export. | `false` |
| `INTEGRITY_SIM_HARNESS_ENABLED` | Enables simulation dataset adapters. | `false` |
