# Science Datasets

This document describes the datasets used in the Summit Science Pack.
Manifests are stored in `data/manifests/science/`.

## The Well
- **Source**: Polymathic AI
- **Size**: ~15TB
- **Type**: Physics simulations
- **Access**: External connectors (no raw data in repo)

## AION-1 Surveys
- **Sources**: SDSS, Gaia, etc.
- **Type**: Multimodal astronomy data (imaging, spectra, scalar)
- **Access**: External connectors
