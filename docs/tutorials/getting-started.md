# Getting Started with Summit/IntelGraph

Welcome to the IntelGraph Summit documentation! This guide will help you get up and running with the platform.

## Prerequisites

- Node.js 18+
- Docker & Docker Compose
- pnpm

## Installation

1. **Clone the repository:**

   ```bash
   git clone https://github.com/intelgraph/intelgraph.git
   cd intelgraph
   ```

2. **Bootstrap the environment:**

   ```bash
   make bootstrap
   ```

3. **Start the development stack:**

   ```bash
   make up
   ```

4. **Run Smoke Tests:**

   ```bash
   make smoke
   ```
