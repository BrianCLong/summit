# Closed PR Recap

This directory contains a recap of all closed pull requests in the repository.
It was generated by the Codex CLI to inventory, summarize, and validate the inclusion of all work.

## How to Update

To update this recap, run the recapture process again.

## Salvage Patches

The `artifacts/closed-pr-recapture/unmerged-patches` directory contains patches for closed but unmerged PRs that may be salvageable. Each patch is accompanied by a `.meta.json` file that provides context and a classification for re-application.
