## Diligence Principles (Non-Negotiable)

- **Truthful**: no aspirational claims; only verifiable artifacts.
- **Consistent**: cross-referenced with existing governance and GA documentation.
- **Risk-aware**: material risks are visible and framed with mitigation status.
- **Narrative-aligned**: product positioning matches documented scope.
- **Audit-grade**: evidence pointers are recorded for later verification.
