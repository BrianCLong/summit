# Internal Comms Rules (M&A / Funding)

## Confidentiality

- **Need to Know:** Deal information is strictly limited to the "Deal Team" and designated stakeholders.
- **Project Names:** Use code names for all potential transactions (e.g., Project Summit).
- **No Speculation:** Do not discuss rumors or speculation about the company's future with anyone, internal or external.

## External Communications

- **Media/Press:** Direct all inquiries to the CEO or designated Communications Lead. "No comment" is the only authorized response.
- **Social Media:** Do not post about potential deals or unusual activity.

## Data Handling

- **Data Room Only:** Do not share diligence documents via email, Slack, or Google Drive links. All sharing must occur via the secure Virtual Data Room (VDR).
- **Watermarking:** Sensitive documents should be watermarked where possible.
- **Access Control:** Access to the VDR is granted on a strict least-privilege basis and audited.
