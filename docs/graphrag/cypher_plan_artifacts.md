# Cypher Plan Artifacts (Summit)

## Purpose
Track the cost and complexity of Cypher plans generated by retrievers over time, visually representing this via a heatmap, and outputting JSON files for machine readability.

## Flow
1. **Extraction**: `tools/cypher/plan_extract.mjs` extracts plan summaries from retrieval outputs (JSON or JSONL).
2. **Rendering**: `tools/cypher/plan_heatmap.py` parses these plan summaries to generate an SVG heatmap and JSON artifacts.
3. **CI integration**: `.github/workflows/cypher-plan-heatmap.yml` extracts the plans and renders the heatmap on every CI run, uploading the SVG and JSON artifacts.
