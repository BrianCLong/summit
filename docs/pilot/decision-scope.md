### In-Scope Decision Classes (Choose 3–5)

| Decision Class               | Description                           |
| :--------------------------- | :------------------------------------ |
| Incident Containment         | Isolating systems, disabling access   |
| Automated Response           | Allow / block scripted remediation    |
| Escalation Threshold         | SOC → Exec escalation                 |
| Data Exposure Response       | Disclosure, notification, containment |
| Privileged Access Revocation | Emergency credential invalidation     |

### Out of Scope (Explicit)

- Alert triage
- Low-risk automation
- Reporting dashboards
- Non-security workflows

> **Rule:** Any decision not listed here is automatically excluded.
