### Valid Refusal Reasons

- Insufficient evidence
- Authority mismatch
- Policy violation
- Confidence below threshold
- Conflicting signals unresolved

### Refusal Guarantees

- Refusal is **not** failure
- Refusal is auditable
- Refusal does not escalate by default
- Refusal protects operators

### Required Refusal Output

- Refusal ID
- Applicable rule
- Evidence gap summary
- Next admissible action
