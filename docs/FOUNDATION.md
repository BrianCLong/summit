# Summit Foundation

**Scope:** The base layer of the Summit Intelligence Platform.
**Focus:** Data unification, Knowledge Graph construction, and core API access.

## Capabilities
- **Universal Ingestion:** Connectors for structured and unstructured data (PDFs, Emails, Databases).
- **Knowledge Graph:** Dynamic entity resolution and relationship mapping.
- **Semantic Search:** Natural language querying of the enterprise memory.

## Architecture
- **Infrastructure:** Neo4j (Graph), Postgres (Relational), Redis (Cache).
- **API:** GraphQL and REST endpoints for data access.
