# PR 01: Schemas for InvestigationRun + EvidenceObject

## Summary
Implements deterministic schemas for the core Subsumption primitives (`InvestigationRun`, `EvidenceObject`, `EvidenceBundle`) and adds a schema validation gate with fixtures.

## Gates Affected
- `gate:schema:validate` (via `scripts/gates/validate_schemas.mjs`)

## How to run locally
Run the schema validation gate:
```bash
node scripts/gates/validate_schemas.mjs
```

## Fixture Notes
- Located in `fixtures/schemas/valid/` and `fixtures/schemas/invalid/`.
- Valid fixtures are used to ensure definitions are loose enough for expected usage but tight enough to be useful.
- Invalid fixtures intentionally omit required fields to trigger rejection.
