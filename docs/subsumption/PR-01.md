# PR 01: InvestigationRun and EvidenceObject Schemas

**Branch:** `subsumption/pr01-schemas-run-evidence`

## Summary
Adds canonical JSON schemas for `InvestigationRun`, `EvidenceObject`, and `EvidenceBundle` primitives. Includes deterministic design enforcement (no timestamps, stable ordering required).
Adds fixture-based CI validation for schemas.

## Gates
- **`gate:schema:validate`**: Added `scripts/gates/validate_schemas.mjs` script
- Integrated into `.github/workflows/pr-gate.yml` via `pnpm gate:schema`

## How to run locally
Run the following command to validate the schemas against the fixtures:

```bash
pnpm gate:schema
```
