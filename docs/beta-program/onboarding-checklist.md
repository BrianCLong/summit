# Pilot Onboarding Checklist

## Phase 1: Preparation
- [ ] **Contract**: Signed Beta Addendum.
- [ ] **Use Case**: Defined success criteria and "Time-to-Value" metric.
- [ ] **Team**: Identified Customer Champion and Technical Lead.

## Phase 2: Configuration
- [ ] **Allowlist**: Added Customer Tenant ID to feature flags.
- [ ] **Quotas**: Adjusted rate limits/quotas for pilot usage.
- [ ] **Monitoring**: Enabled "Pilot Dashboard" for the account.

## Phase 3: Kickoff
- [ ] **Welcome Call**: Walkthrough of features and known limitations.
- [ ] **Support**: Shared escalation channel (e.g., Shared Slack).
- [ ] **Cadence**: Scheduled weekly feedback session.

## Phase 4: Active Pilot
- [ ] **Weekly**: Check usage stats and open issues.
- [ ] **Mid-Point**: Review progress against success criteria.

## Phase 5: Conclusion
- [ ] **Review**: Exit review (Expand/Renew/Fix).
- [ ] **Reference**: Request case study/testimonial.
