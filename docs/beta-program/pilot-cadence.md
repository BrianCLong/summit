# Pilot Cadence

## Weekly Check-in
- **Duration**: 30 mins
- **Attendees**: PM, Eng Lead, Customer Champion
- **Agenda**:
    1. Review Open Issues
    2. Review Usage Metrics (Pilot Dashboard)
    3. Roadmap Alignment
    4. Feedback collection

## Issue Review
- Occurs bi-weekly or as needed.
- Review critical bugs and feature requests.
