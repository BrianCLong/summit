# Reference Pipeline

## Eligibility
- Successful pilot (Success Criteria met as per Onboarding Checklist).
- Positive NPS/CSAT score (>8).
- Customer has realized demonstrable value (Time-to-Value metric met).

## Process
1. **Request**: Account Manager requests permission for logo use and quote.
2. **Draft**: Marketing drafts case study (Challenge, Solution, Outcome).
3. **Approve**: Customer approves content.
4. **Publish**: Add to website and sales deck.

## Incentives
- Discount on first year GA pricing for public reference.
