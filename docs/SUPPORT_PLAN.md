# GA Support Plan

## Support Tiers

### L1 (General Support) - "The Shield"

- **Who**: Community Managers / Junior Support.
- **Scope**: Password resets, "How do I?", Billing questions.
- **Routing**: If technical error -> Escalate to L2.
- **Response Time**: < 24 Hours.

### L2 (Technical Support) - "The Fixers"

- **Who**: Support Engineers.
- **Scope**: Reproduction of bugs, Log analysis, Config issues.
- **Routing**: If code fix needed -> Escalate to L3 (Eng).
- **Response Time**: < 8 Hours.

### L3 (Engineering) - "The Builders"

- **Who**: Product Engineers (Rotation).
- **Scope**: Code fixes, Sev-1/2 Incidents, Data Patching.
- **Response Time**: < 4 Hours (Sev-2), < 1 Hour (Sev-1).
