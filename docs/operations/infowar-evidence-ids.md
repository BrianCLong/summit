# Info Warfare Evidence ID Registry

This document serves as the single source of truth for reserved Evidence IDs.

## ID Format
`EVD-INFOWAR-<AREA>-<NNN>`

- `AREA`: `EVD`, `NARR`, `REG`, `OPS`, `ACTMEAS`
- `NNN`: 001-999

### AREA: REG (Regulation)
