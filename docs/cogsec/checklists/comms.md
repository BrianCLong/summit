# Cognitive Security Comms Checklist

- [ ] **Situation Awareness**:
    - [ ] Review Triage Report and Artifacts.
    - [ ] Understand the core false narrative.
- [ ] **Strategy Selection**:
    - [ ] Evaluate "Strategic Silence" vs. "Active Response".
    - [ ] Consult with Legal on liability/risk.
- [ ] **Drafting**:
    - [ ] Select appropriate template (Holding Statement / FAQ).
    - [ ] Draft internal comms (employees need to know first).
    - [ ] Draft external comms (social, press, partners).
- [ ] **Approval**:
    - [ ] Get sign-off from Incident Commander.
    - [ ] Get sign-off from Legal.
- [ ] **Dissemination**:
    - [ ] Publish to selected channels.
    - [ ] Monitor reaction/sentiment.
