# Cognitive Security Legal Checklist

- [ ] **Rights Impact Assessment**:
    - [ ] Review proposed actions for privacy violations.
    - [ ] ensure no individual profiling is occurring.
    - [ ] Verify "deny-by-default" policy is respected.
- [ ] **Liability Review**:
    - [ ] Assess potential defamation or trademark infringement claims.
    - [ ] Review platform Terms of Service for takedown grounds.
- [ ] **Regulatory Compliance**:
    - [ ] Check if the incident triggers disclosure requirements (e.g., SEC, GDPR).
    - [ ] Advise on preservation of evidence (Legal Hold).
- [ ] **Vendor Management**:
    - [ ] If using external vendors for remediation, ensure they comply with rights policies.
