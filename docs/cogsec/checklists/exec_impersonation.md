# Executive Impersonation Checklist

- [ ] **Verification**:
    - [ ] Contact the Executive or their Chief of Staff immediately.
    - [ ] Confirm location and activity at the time of the alleged recording.
    - [ ] Analyze media for deepfake artifacts (audio glitches, unnatural blinking).
- [ ] **Containment**:
    - [ ] Notify internal security teams (prevent whaling/phishing).
    - [ ] Notify board members if material logic.
- [ ] **Platform Action**:
    - [ ] Report to hosting platform under "Impersonation" policy.
    - [ ] Escalate through trusted partner portals if available.
- [ ] **Public Response**:
    - [ ] Issue immediate denial if widespread.
    - [ ] Use "Verification Script" template.
