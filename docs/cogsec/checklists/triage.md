# Cognitive Security Triage Checklist

- [ ] **Signal Source Verification**:
    - [ ] Is the source internal (employee) or external (social/news)?
    - [ ] If external, is the account verified or known?
- [ ] **Artifact Collection**:
    - [ ] Capture screenshots/URLs.
    - [ ] Download media (video/audio) if safe.
    - [ ] Archive content using standard tools (Wayback Machine or internal).
- [ ] **Initial Verification**:
    - [ ] Does the content contradict known facts?
    - [ ] Are there signs of manipulation (glitches, inconsistencies)?
    - [ ] Verify with subject matter experts (e.g., did the exec actually give a speech?).
- [ ] **Severity Classification**:
    - [ ] SEV-1: Critical (Safety/Financial).
    - [ ] SEV-2: High (Brand/Reputation).
    - [ ] SEV-3: Low (Noise).
- [ ] **Notification**:
    - [ ] Notify Incident Commander.
    - [ ] Open `CognitiveIncident` ticket.
