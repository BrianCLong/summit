# Integration Tiers

Summit exposes **three concentric ecosystem layers**, each with different rights.

## Tier 1: Insight Providers (Safest, Broadest)
*   **Role**: Read-only value creation.
*   **Who**: Threat intel vendors, Risk scoring services.
*   **Rights**: Provide signals, add annotations.

## Tier 2: Recommendation Engines (High Value, Controlled)
*   **Role**: Propose, never decide.
*   **Who**: ML models, Scenario planners.
*   **Rights**: Propose actions, Rank alternatives.

## Tier 3: Tool Integrations (Execution Adjacent)
*   **Role**: Execution only after approval.
*   **Who**: SOAR tools, Infra automation.
*   **Rights**: Execute signed commands, Report outcomes.
