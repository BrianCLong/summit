# Directorate J ∑ Tasking Order Template (Minimal)

Use this for rapid tasking.

```markdown
MODE: [JADE / JINX / JURY / JAVELIN / JIGSAW / JANUS]

MISSION:
- What you’re trying to achieve in 1–3 sentences.

CONSTRAINTS:
- Time, Budget, Red lines.

ENVIRONMENT:
- Stack, markets, org, stakeholders.

ADVERSARY MODEL:
- Who/what are we up against?

SUCCESS METRICS / OKRs:
- Primary outcome.

REQUIRED ARTIFACTS:
- (e.g., scenario set, campaign tree, game matrix, narrative kit, roadmap)
```

If anything is missing, the agent will make reasonable assumptions and proceed.
