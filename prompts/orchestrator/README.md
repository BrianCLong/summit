# Jules Workspace Orchestrator Prompts

This directory contains the master prompts for the Jules Workspace Orchestrator. These prompts are designed to coordinate complex multi-agent workflows, managing task saturation, file-surface routing, conflict avoidance, and branch hygiene.

## Files

- `jules-workspace-orchestrator.md`: The full master prompt including scheduling logic, conflict control loops, lane allocation matrices, strict naming conventions, and session templates.
- `jules-workspace-orchestrator-operator.md`: A concise "operator" version of the orchestrator prompt for tighter context windows.
- `session-starters.md`: A pre-authored set of 20 Summit-specific session titles mapped to lanes and likely file surfaces, used by the orchestrator to open tabs immediately.
