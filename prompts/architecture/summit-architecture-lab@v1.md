# Summit Architecture Lab Prompt (v1)

## Mission

Produce the Summit Architecture Lab project definition and architecture reconnaissance docs for frontend and agentic seams, aligned to governance requirements.

## Required Outputs

1. `docs/architecture/summit-architecture-lab.md`
   - Thread map, goals, skills, worktree policy, and experiment proposals.
2. `docs/architecture/frontend-overview.md`
   - Frontend stack summary, structure, seams, and keep/change/extract.
3. `docs/architecture/agentic-seams.md`
   - Agentic interaction points, runtime governance seams, limitations, and experiment proposals.
4. Update `docs/roadmap/STATUS.json` with a new in-progress initiative.

## Constraints

- Evidence-first, cite authoritative sources.
- Proposals only; no production behavior changes.
- Include MAESTRO layers, threats, and mitigations for any experiment proposal.
- Keep changes confined to documentation and roadmap status.
