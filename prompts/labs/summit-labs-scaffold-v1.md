# Summit Labs Scaffolding Prompt (v1)

## Objective

Create repository scaffolding for Summit Labs, including experiment templates,
research preview specs, and promotion gates. Update the roadmap status to
record the Labs track scaffolding initiative.

## Required Outputs

- `labs/README.md`
- `labs/experiment-template.md`
- `labs/research-preview-spec.md`
- `labs/promotion-gates.md`
- Update `docs/roadmap/STATUS.json` with a Labs initiative entry and timestamp.

## Constraints

- Keep changes within documentation and governance scaffolding only.
- Use policy-as-code language for governance controls.
- Ensure promotion gates include evaluation, safety, reliability, and
  enterprise criteria.
