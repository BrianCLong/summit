# Prompt: Narrative ITEM Integration Plan (v1)

## Objective

Create a governed documentation update that records the ITEM integration plan for narrative frame graphs and influence strategy taxonomy, and update roadmap status notes to reflect the documentation change.

## Required outputs

1. `docs/narrative/item-integration-plan.md` with:
   - readiness assertion reference to `docs/SUMMIT_READINESS_ASSERTION.md`
   - present state + mandated future state
   - PR1–PR6 stack summary
   - MAESTRO alignment (layers, threats, mitigations)
   - deterministic evidence bundle standard
2. `docs/roadmap/STATUS.json` updated with:
   - refreshed `last_updated`
   - revision note summarizing the documentation change
   - narrative initiative note referencing the ITEM integration plan

## Constraints

- Evidence-first, deterministic language.
- No actor attribution claims.
- Follow repository governance and documentation conventions.
- Keep scope limited to narrative docs + roadmap status.

## Acceptance criteria

- Documentation is deterministic, structured, and anchored to governance references.
- Roadmap status reflects the new narrative documentation plan.
- Changes are limited to the declared scope.
