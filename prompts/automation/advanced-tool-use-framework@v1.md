# Prompt: Advanced Tool Use Framework (Automation Docs)

## Goal

Document the Summit prompt framework for advanced tool use in automation workflows and
update automation documentation to reference it.

## Scope

- Documentation under `docs/automation/`
- Roadmap status updates in `docs/roadmap/STATUS.json`
- Prompt registry updates in `prompts/registry.yaml`
- Task specs recorded under `agents/examples/`

## Requirements

- Add or update the advanced tool use prompt framework documentation.
- Link the framework from `docs/automation/GUIDE.md`.
- Update `docs/roadmap/STATUS.json` with an accurate revision note and timestamp.
- Maintain governance alignment and traceability.

## Output Expectations

- Clear documentation with Summit governance framing.
- Minimal, scoped changes limited to the declared paths.
