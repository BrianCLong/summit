# GA Release Captain Skill

This skill guides the release captain through the GA release process.

## Rules

1.  **Produce GA Evidence Bundle artifacts.**
    *   Ensure all evidence is collected and bundled.

2.  **Update GA Readiness dashboards.**
    *   Ensure dashboards reflect the current state of the release.

3.  **Confirm branch protection requirements.**
    *   Verify that `main` and release branches have appropriate protection rules.

4.  **Output merge-train readiness status.**
    *   Verify that the merge queue is healthy and processing PRs.
