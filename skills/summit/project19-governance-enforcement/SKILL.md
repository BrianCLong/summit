# Project 19 Governance Enforcement Skill

This skill enforces governance requirements for Project 19.

## Rules

1.  **Ensure PR references an issue/epic.**
    *   All PRs must link to a tracking issue.

2.  **Ensure required fields (risk, evidence, owner) exist.**
    *   Governance artifacts must have all required metadata.

3.  **Refuse “TODO placeholders” in governance artifacts.**
    *   Governance documents must be complete.
