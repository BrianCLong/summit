# Summit Examples

This directory contains examples and templates for extending and integrating with the Summit platform.

## Directory Structure

- **plugins/**: Example plugins for extending core functionality.
- **pipelines/**: Custom data ingestion and processing pipelines.
- **curl/**: Example API requests.
- **dashboard-templates/**: JSON templates for Grafana dashboards.
- **runbooks/**: Operational runbooks.

## Getting Started

To use these examples, copy the relevant files to your project workspace or custom plugins directory.

### Running Python Examples

```bash
pip install summit-sdk
python pipelines/custom-ingest.py
```
