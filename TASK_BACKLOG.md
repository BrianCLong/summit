# TASK_BACKLOG.md - AI Agent Task Backlog

This document tracks the prioritized list of tasks for AI coding agents. Agents should only work on tasks from this backlog.

## Task Status Lifecycle

- **Queued**: The task is ready to be worked on.
- **Planned**: An agent has created a plan for the task in `AGENT_PLANS.md`.
- **In Progress**: The task is actively being worked on.
- **Review**: The task is complete and a pull request has been submitted for review.
- **Done**: The pull request has been approved and merged.
