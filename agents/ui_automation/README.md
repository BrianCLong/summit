# UI Automation Agent Module

This module provides secure, deterministic UI automation capabilities for Summit agents.

## Status

**Experimental** - Under active development.
Feature Flag: `FEATURE_UI_AUTOMATION` (Default: OFF)

## Architecture

- **Runner**: Deterministic browser automation (Playwright).
- **Policy**: Strict allow/deny for domains and actions.
- **Evidence**: JSON artifacts for every run.
