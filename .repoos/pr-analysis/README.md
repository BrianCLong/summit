# PR Architecture Copilot Outputs

This directory is generated by `scripts/pr-architecture-copilot.mjs`.

Artifacts:

- `pr-change-summary.json`: file/module/subsystem impact summary
- `dependency-impact.json`: dependency additions, circular risk, fan-out impact
- `architecture-boundaries.json`: cross-subsystem boundary checks
- `risk-score.json`: deterministic structural risk score and CI failure delta
- `pr-comment.md`: GitHub-ready architecture comment
- `pr-architecture-report.html`: browser-friendly architecture impact report

Run locally:

```bash
node scripts/pr-architecture-copilot.mjs --files-json .repoos/pr-analysis/pr-files.json
```
