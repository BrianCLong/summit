## 2025-05-23 - Destructive "Quick Actions"
**Learning:** "Quick Prompts" that replace user input without confirmation are a data loss risk. Users treat "chips" as additives, not replacements, unless explicitly styled as tabs/templates.
**Action:** When using suggestion chips, check if input is non-empty. Append or ask for confirmation, or use a "template" visual metaphor if replacement is intended.
