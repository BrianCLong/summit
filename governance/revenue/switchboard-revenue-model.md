# Switchboard Revenue Model

## Positioning
Consumer OSINT front door that feeds Summit enterprise pipeline.

## Pricing Tiers

### Free
- Basic entity lookup.
- Limited graph view.
- 5 searches/day.

### Pro ($29-$79/month)
- Full graph exploration.
- Report export.
- Claim verification.
- Alerts.

### Team ($299+/month)
- Shared investigations.
- API access.
- Bulk search.

### Enterprise
- Summit backend integration.
- Private deployment.
- Custom policies.

## Revenue Flywheel
Switchboard -> data capture -> Summit insights -> enterprise upsell.
