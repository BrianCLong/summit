## 2024-03-01 - Avoid inline array operations in large React component renders
**Learning:** React components containing inline array operations like `.filter().length` recalculate and allocate new arrays on every render cycle, which causes noticeable GC pressure and jank in highly interactive, data-heavy dashboards (like SIGINT or Forensics dashboards with large array sizes).
**Action:** Use `useMemo` in combination with single-pass `.reduce()` or `for` loops to derive lengths and stats from lists without allocating unnecessary intermediate arrays. Apply this universally to data-heavy React components.
