# Narrative Simulator UI

## Purpose

Allow analysts to review ANFIS coordination fingerprints and intervention simulations.

## Core Elements

- Fingerprint summary (burstiness, similarity, hub concentration).
- Intervention plan list with ROI metrics.
- Counterfactual spread charts (baseline vs. modified).

## Actions

- Select intervention plan and rerun simulation (budget permitting).
- Export attribution artifact and witness chain.
