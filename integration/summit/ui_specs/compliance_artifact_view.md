# UI Spec: Compliance Artifact View (PQLA)

## Purpose

Display proof-carrying analytics outputs with policy and disclosure metadata.

## Layout

- **Output Summary:** aggregated results and disclosure transforms.
- **Policy Panel:** policy decision ID, constraints, determinism token.
- **Attestation Panel:** sandbox attestation status.
- **Audit Links:** witness ledger and transparency log references.

## UX Requirements

- Provide copyable compliance artifact bundle.
- Highlight disclosure adjustments.
