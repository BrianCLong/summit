# UI Spec: Feed Drift Dashboard (FASC)

## Purpose

Surface feed calibration health, drift status, and quarantine actions for analysts and admins.

## Layout

- **Feed List:** reliability score, weight, drift indicator.
- **Drift Detail Panel:** divergence metrics and dwell time.
- **Quarantine Actions:** current policy effect, justification witness.

## UX Requirements

- Provide alerts when drift exceeds thresholds.
- Show calibration artifact references for audit.
- Allow export of transparency log proofs.
