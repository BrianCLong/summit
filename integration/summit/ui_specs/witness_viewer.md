# Witness Viewer UI

## Purpose

Render witness chain entries and validate hash continuity for audit review.

## Core Elements

- Session summary: session ID, start/end time, anchor hash.
- Timeline of steps with inputs/outputs and policy decision IDs.
- Determinism token display with replay links.

## Actions

- Validate chain integrity.
- Export witness chain as JSON.
- Jump to associated evidence bundle or artifact.
