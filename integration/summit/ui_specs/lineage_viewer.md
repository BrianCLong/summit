# Lineage Viewer

## Purpose

Visualize narrative lineage paths with stance drift overlays.

## Core Views

- Lineage graph with quote/remix/link edges.
- Stance drift timeline per narrative cluster.
- Replay token and capsule metadata panel.

## UX Requirements

- Filters for time window, platform, and stance polarity.
- Path highlighting for coordination indicators.
- Export panel for capsule evidence.
