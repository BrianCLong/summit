# UI Spec: Transform Receipt View (SATT)

## Purpose

Show transform execution outputs with license receipts and attestation status.

## Layout

- **Transform Output Panel:** redacted outputs and disclosure summary.
- **License Receipt Panel:** usage counters, measurement hash, policy ref.
- **Attestation Panel:** signer identity and validation status.

## UX Requirements

- Alert when license budget is low.
- Provide download of receipt and witness proofs.
