# ICD-RAM API Outline (OA/MOSA Packaging)

## Endpoints

- `POST /icd-ram/v1/packages`
- `GET /icd-ram/v1/packages/{packageId}`
- `GET /icd-ram/v1/packages/{packageId}/conformance`
- `GET /icd-ram/v1/packages/{packageId}/rights`

## Core Schemas

- `ArchitectureSpec`
- `OaMosaPackage`
- `ConformanceSuite`
- `RightsAssertionArtifact`
