# VCEC API Outline (Experiment Capsules)

## Endpoints

- `POST /vcec/v1/capsules`
- `GET /vcec/v1/capsules/{capsuleId}`
- `POST /vcec/v1/capsules/{capsuleId}/run`
- `GET /vcec/v1/capsules/{capsuleId}/runs/{runId}`

## Core Schemas

- `CapsuleSpec`
- `DeterminismManifest`
- `WitnessChain`
- `ReplayToken`
