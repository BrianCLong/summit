# Evidence Service

## Responsibilities

- Generate releasability packs, assessment bundles, and incident packets.
- Bind outputs to replay tokens and manifests.

## Inputs

- Narrative or telemetry payload
- Scope token
- Policy decision ID

## Outputs

- Evidence artifact bundle
- Replay token
- Manifest hash
