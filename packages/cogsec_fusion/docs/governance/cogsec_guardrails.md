# Cognitive Security Guardrails

## Policy Overview
This module enforces "No Harmful Automation" and "Deny-by-Default" policies for influence operations.

## Active Constraints
- **POLICY-NO-OFFENSIVE-IO**: Do not generate or deploy offensive influence content.
- **POLICY-PROTECT-VULNERABLE**: Flag and block content exploiting vulnerable populations.
- **POLICY-PROVENANCE-REQUIRED**: All disseminated artifacts must have a provenance record.

## Legal Norms
- **ICRC**: Human brain should not become a battlefield.
- **NATO**: Cognitive Warfare Concept framing.
