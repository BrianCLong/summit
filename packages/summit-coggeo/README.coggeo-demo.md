# Cognitive Geophysics Demo

This package provides a laptop-local demo for:

- ingest → extract → cluster → terrain → storm detection
- DuckDB + Parquet-backed terrain storage
- Express endpoints for narratives / storms / explain / terrain tiles
- Map-ready MVT + GeoJSON fallback

## Quickstart

```bash
cd packages/summit-coggeo
npm install
npm run coggeo:demo
```
