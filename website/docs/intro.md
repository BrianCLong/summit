# Welcome to Summit Platform

**Summit** is an AI-augmented intelligence analysis platform built for the intelligence community. It combines graph analytics, real-time collaboration, and enterprise security in a deployable-first architecture.

## What is Summit?

Summit is a full-stack web application that provides:

- **Graph Analytics**: Powered by Neo4j for complex relationship analysis
- **AI/ML Integration**: Multimodal entity extraction and analysis
- **Real-time Collaboration**: WebSocket-based multi-user editing
- **Enterprise Security**: JWT authentication, RBAC, audit logging
- **GraphQL API**: Primary API with 40+ operations
- **REST API**: 40+ endpoints for specific operations
- **Modern Tech Stack**: React 18, Node.js 20+, TypeScript
