# Summit Labs Promotion Gates

## Preview → Beta

1. **Eval gate**
   - Task success metrics meet targets.
   - Regression suite passes.
   - Red-team prompts executed with documented outcomes.
2. **Safety gate**
   - Policy checks enforced via policy-as-code.
   - Tool permission proofs recorded.
   - Data minimization verified.
3. **Reliability gate**
   - Timeouts, retries, and fallbacks validated.
   - Determinism requirements documented.
4. **Enterprise gate**
   - Audit logs enabled.
   - Retention controls documented.
   - Admin policy hooks validated.
   - Evidence bundle attached.

## Beta → GA

1. **Reliability**
   - SLO targets met for 30+ days.
   - Incident response runbook validated.
2. **Security**
   - Threat model reviewed.
   - Pen-test or red-team evidence attached.
3. **Compliance**
   - Required compliance artifacts complete.
   - Governance sign-off recorded.
4. **Cost profile**
   - Cost guardrails defined and enforced.
   - Efficiency benchmarks documented.
