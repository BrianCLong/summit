# Documentation

This directory contains all documentation for the SummitCogWar project, including technical specifications, case studies, and user guides.

## Key Documents

- **Case Studies:** In-depth analyses of historical and simulated cognitive warfare events, such as the "2025 Election CogWar" simulation.
- **Technical Architecture:** Detailed diagrams and explanations of the system's architecture.
- **User Manuals:** Guides for using the various components of the platform.
- **MVP Checklist:** A running list of features and capabilities required for the Minimum Viable Product release.
