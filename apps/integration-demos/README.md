# Summit Integration Demos

This package contains demo descriptors for the Integration Magnet Program.

## Demos

- `ci-pipeline-demo.json`: CI pipeline integration flow.
- `architecture-analysis-demo.json`: Architecture analysis workflow.
- `observability-metrics-demo.json`: Observability ingestion workflow.

Run with your preferred orchestrator by reading each JSON file and mapping the `steps` sequence to your local environment.
