# NATO INFOSEC Baseline Mapping

## Baseline Controls

- Identity and access management for coalition scopes
- Audit logging for all data exports and incident packets
- Network egress restrictions per scope

## Evidence Artifacts

- Scope token issuance logs
- Egress receipts with byte/entity budgets
- Transparency log digests

## Policy-as-Code

- Scope token validation gates exports and egress.
- Egress budgets enforced via policy before outbound transfers.
