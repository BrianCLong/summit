# Evaluator Handoff Checklist

Use this checklist before delivering artifacts to evaluators.

## Contract Compliance

- [ ] Interface methods implement evaluator contract.
- [ ] Determinism token schema validated.
- [ ] Conformance suite passes in evaluator mode.

## Artifact Integrity

- [ ] Metric proof object includes Merkle commitments.
- [ ] Witness chain is hash-linked end-to-end.
- [ ] Transparency log receipt attached.

## Rights & Disclosure

- [ ] Rights assertion artifact includes government rights categories.
- [ ] Proprietary stubs labeled and tested.
- [ ] Policy-as-code enforcement logs recorded.
