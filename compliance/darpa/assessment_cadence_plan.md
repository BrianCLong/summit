# Assessment Cadence Plan

This plan aligns evaluator runs to recurring assessment cycles.

## Cadence

- **Mini assessments**: Every 6 months using frozen dataset snapshots.
- **Full assessments**: Annual evaluations with full conformance suite and OA/MOSA
  package review.

## Required Outputs

- Determinism tokens for each run
- Proof objects and witness chains
- Transparency log receipts
- Rights assertion updates (if applicable)
