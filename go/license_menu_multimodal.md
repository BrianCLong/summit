# License Menu: Multimodal Frontier

## Components
- **Multimodal Frontier SDK**: APIs, schemas, ingestion pipelines, and client libraries.
- **Multimodal Agent Runtime**: Runtime orchestration, tool adapters, and governance controls.
- **Vertical Assistants**: Document QA, Chart QA, and UI automation packs.

## Licensing Options
1. **SaaS**: Usage-based; includes hosted detectors (OCR/table/chart) and managed IntelGraph storage.
2. **Enterprise On-Prem**: Annual subscription with self-hosted runtimes, BYO detectors, and air-gapped governance mode.
3. **OEM/Embedded**: Per-seat or per-application royalty for SDK embedding; optional revenue share for tool marketplace.
4. **Eval/Developer Tier**: Limited tokens per month; includes sandbox telemetry for curriculum feedback.

## Add-ons

- Compliance pack (redaction, PII filters, audit exports).
- Performance pack (GPU autoscaling, caching tier for layouts/embeddings).

## SLAs & Support
- Tiered SLOs for ingestion latency, tool invocation success, and answer accuracy.
