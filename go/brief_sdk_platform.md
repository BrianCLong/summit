# SDK Platform Brief

## Offering
- **Summit SDK Core (OSS-friendly)**: client, flows, tools, RAG helpers, local transport, telemetry hooks.
- **Premium Extensions**: governance ledger integration, SRE telemetry exporters, enterprise policy packs.
- **Blueprint Packs**: verticalized graph templates for doc copilot, ops agent, and dev console.
- **Hosted Dev Platform**: managed runtime, deployment/versioning, policy management, dashboards.

## Target Customers
- Enterprises building internal copilots on Summit models.
- ISVs integrating Summit as an agentic platform.
- Cloud providers seeking Summit-style DX on their infra.

## Pricing Levers
- Per-seat developer licenses for premium SDK modules.
- Per-tenant/runtime-hour for hosted platform.
- Blueprint pack subscriptions with vertical updates.

## GTM Hooks
- Fast start via CLI scaffolding and telemetry defaults.
- Migration-safe APIs compatible with local and hosted models.
- Policy-first messaging for compliance-heavy sectors.
