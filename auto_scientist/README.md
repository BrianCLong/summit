# Auto-Scientist Stack

This directory contains the implementation of the Auto-Scientist Stack, including the Frontier Alignment & Oversight Loop.

## Structure

- `spec/`: Specifications and architecture documents.
- `ip/`: Intellectual Property artifacts, prior art tracking, and patent drafts.
- `impl/`: Implementation code (Python).
- `experiments/`: Experiment scripts and configs.
- `go/`: Commercialization materials.

## Frontier Alignment & Oversight Loop (v0.1)

The Frontier Alignment & Oversight Loop aims to build a DPO/RLHF-style alignment stack with a model-in-the-loop oversight layer.
