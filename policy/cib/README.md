# Coordinated Inauthentic Behavior (CIB) Policy Intent

This folder holds policy-as-code governing CIB detection gates, grounded in the research signals
and evaluation hooks in `docs/research/NARRATIVE_IO_READING_LIST.md` and
`docs/research/NARRATIVE_IO_REFERENCE_TO_PIPELINE_MAP.md`. Policies here should encode explicit
coordination evidence thresholds, authenticity checks, and audit-ready explanations for analyst
review workflows.
