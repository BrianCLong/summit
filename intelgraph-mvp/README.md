# IntelGraph MVP

This repository contains the Minimum Viable Product for the IntelGraph platform.

## Services

- **Gateway:** GraphQL API endpoint.
- **Ingest:** Data ingestion workers.
- **Infra:** Service configurations (database schemas, mappings).
- **Docs:** Project documentation.
- **Scripts:** Helper and automation scripts.

## Getting Started

To bring up the development stack, run:

```bash
make up
```
