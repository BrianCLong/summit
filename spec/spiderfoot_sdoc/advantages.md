# SDOC Advantages

## Strategic Advantages

- **Selective disclosure** replaces raw data dumps.
- **Sensitivity budgets** enforce privacy constraints by default.
- **Capsule ledger** provides verifiable audit trails for OSINT scans.

## Operational Advantages

- Passive-only default for unauthorized active probing.
- Retention TTLs enforce data minimization.
- Counterfactual capsule outputs quantify information loss.
