# OCT Spec Overview

Defines the Recorded Future wedge for Outcome-Calibrated Threat Signal Portfolio.

## Goals

- Build risk portfolios decomposed into factor components.
- Calibrate factors against observed outcomes with backtest proofs.
- Output risk portfolio artifacts with replay tokens and policy bindings.

## Inputs

- Entity identifier for a cyber object.
- Threat signals from multiple sources with provenance.
- Outcome data for calibration.

## Outputs

- Factor portfolio with calibrated weights.
- Backtest artifacts with statistical measures.
- Replay-bound risk portfolio artifacts.
