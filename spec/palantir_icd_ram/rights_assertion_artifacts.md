# Rights Assertion Artifacts

Defines structured rights tables for OA/MOSA deliverables.

## Table Columns

- Deliverable/component name.
- Interface linkage and proprietary boundary marker.
- Asserted government rights category (e.g., unlimited, government purpose, restricted) with DFARS references.
- Restrictive markings and peer-review allowances.
- Technology insertion notes and evaluator execution stubs when proprietary code withheld.
