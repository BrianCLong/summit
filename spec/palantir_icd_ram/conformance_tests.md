# Conformance Tests

Describes evaluator-facing conformance testing of plug-ins and modules against the ICD.

## Test Coverage

- Interface method signatures and schema validation.
- Effect typing enforcement and egress constraints.
- Deprecation adherence and adapter correctness.
- Proprietary stub behavior validation to ensure evaluator operability.

## Packaging

- Delivered alongside ICD and rights artifacts; includes replay tokens and transparency digests.
- Tests executed by evaluator harness with deterministic seeds.
