# Recorded Future AEC: Assessment Evidence Compiler

Continuous evidence compiler mapping telemetry to NIST SP 800-171 Rev. 3 controls and DFARS/CMMC readiness artifacts.

## Capabilities

- Collects telemetry, maps to control catalog, assembles control-evidence bundles.
- Computes assessment status and DFARS readiness indicators for SPRS posting.
- Generates assessment artifacts with replay tokens and redaction for export.
- Supports counterfactual evidence impacts when telemetry sources are excluded.
