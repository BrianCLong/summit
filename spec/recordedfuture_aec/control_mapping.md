# Control Mapping

Rule set aligning telemetry sources to NIST SP 800-171 Rev. 3 controls.

- **Mapping entries:** control ID, objective, telemetry sources, sufficiency rules, exclusion impact notes.
- **Catalog alignment:** includes DFARS clause cross-references and CMMC affirmation windows.
- **Cache:** per-control evidence cache keyed by replay token for reuse.
