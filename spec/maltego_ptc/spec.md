# PTC Spec Overview

Defines the Maltego wedge for Provenance-Typed Transform Contracts.

## Goals

- Register transforms with typed effect and disclosure declarations.
- Execute transforms with policy checks and sandbox controls.
- Emit proof-carrying artifacts with witness chains and provenance records.

## Inputs

- Transform specification including effect declaration and disclosure constraint.
- Subject context and purpose for policy checks.
- Input entity for execution.

## Outputs

- Transform artifact containing witness and provenance records.
- Redacted outputs honoring disclosure constraints and budgets.
