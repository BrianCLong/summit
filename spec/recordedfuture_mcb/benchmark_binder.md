# MCB Benchmark Binder

The benchmark binder packages contract execution evidence for evaluators.

## Contents

- Metric results with acceptance-threshold comparisons.
- Confidence intervals and methodology description.
- Backtest summaries and historical deltas.
- Drift detector findings and alerts.
- Replay token binding binder to data snapshots and scoring code versions.
- Counterfactual binder summaries excluding selected feeds with reported deltas.
