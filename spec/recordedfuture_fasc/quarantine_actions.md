# Quarantine Actions (FASC)

## Actions

- **Weight reduction:** Reduce influence in fusion pipeline.
- **Exclude:** Temporarily remove from fusion.
- **Corroboration-only:** Route to confirmation path requiring multiple sources.

## Execution

1. Validate policy effect for quarantine action.
2. Apply action and emit witness ledger entry.
3. Log transparency event and compliance decision.
4. Notify feed operators if required by policy.

## Recovery

- Require evidence of recovery before lifting quarantine.
- Recompute feed calibration and log decision.
