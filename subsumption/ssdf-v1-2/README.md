# SSDF v1.2 Subsumption Bundle

This bundle implements alignment with NIST SP 800-218 Revision 1 (IPD) - Secure Software Development Framework (SSDF) Version 1.2.

## Structure

- `manifest.yaml`: The source of truth for this bundle.
- `claim_registry.json`: Registry of SSDF claims/practices/tasks.
- `mapping_registry.json`: Mapping of claims to Summit controls and evidence.

## Governance

This bundle adheres to the Summit Subsumption Bundle standard.
