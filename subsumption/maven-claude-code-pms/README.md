# Maven – Claude Code for PMs (Summit Subsumption)

## Overview
This directory contains the **Summit-style Subsumption Plan** for the Maven course: **"Claude Code for PMs: Data → Decisions Workflows"**.

## Artifacts

- **[manifest.yaml](manifest.yaml)**: Metadata, links, and project status.
- **[claims.md](claims.md)**: Extracted claims and Summit-original subsumption claims.
- **[SOURCE_LEDGER.yml](SOURCE_LEDGER.yml)**: Attribution and license classification for the course material.
- **[INTEGRATION_PLAN.md](INTEGRATION_PLAN.md)**: PR stack plan and mapping to Summit's `innovation` lane.
- **[TRANSCENDENCE.md](TRANSCENDENCE.md)**: Net-new capabilities (Graph-native analysis, Multi-agent consensus) that surpass the source material.

## Summary
The goal of this subsumption is to transition from manual "AI-assisted PM tasks" to a **governed, AI-native PM ecosystem** within the Summit platform. This includes versioned analytical prompts, deterministic decision provenance, and policy-native dashboards.

## Success Criteria
1.  All MAVEN-CLAUDE-PM claims are mapped to Summit integration points.
2.  The `innovation` lane in the PR plan is initialized.
3.  The `DecisionLedger` service is extended to support PM-assisted memos.
