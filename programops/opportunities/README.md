# Opportunities

This directory contains JSON metadata for funding opportunities.

## Format
*   File name: `<id>.json`
*   Content: Public metadata only. No solicitation text.

## Example
```json
{
  "id": "DARPA-PS-26-09",
  "sponsor": "DARPA",
  "vehicle": "ISO",
  "public_url": "https://sam.gov/...",
  "dates": { "posted": "2025-12-19" }
}
```
