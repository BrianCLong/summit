# User Interface: The War Room

This directory contains the source code for the SummitCogWar user interface, a Streamlit-based "war room" that provides a centralized command and control center for all cognitive warfare operations.

## Key Features

- **Red CogSwarm Control:** Launch, monitor, and manage offensive memetic campaigns.
- **Blue MindShield Dashboard:** Visualize real-time cognitive threats and deploy defensive countermeasures.
- **Cognition Legions Explorer:** Interact with the simulation environment, run scenarios, and analyze results.
- **Neuro-Ethics Console:** Review and approve HITL escalations and monitor ethical performance metrics.
