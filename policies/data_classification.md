# Data Classification Policy

## Levels

- **Public**: Open data, no restrictions.
- **Internal**: Internal use only, no external sharing.
- **Restricted**: PII, credentials, or sensitive IP. Access on need-to-know basis.
- **Science**: External scientific datasets. Subject to specific license restrictions (see manifests).

## Science Data Handling

- **No Raw Export**: Science data should not be exported raw without approval.
- **Attribution**: Must attribute sources as per manifest.
- **Version Pinning**: All datasets must be pinned to specific versions in manifests.
