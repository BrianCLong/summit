# Evidence Bundle Schema

This directory contains the JSON Schema definitions for the Evidence Bundle.

## Versioning

We use Semantic Versioning (Major.Minor.Patch) for the schema.

- **Major**: Breaking changes (renaming required fields, removing fields).
- **Minor**: Backward-compatible changes (adding optional fields).
- **Patch**: Corrections that do not affect validation logic (description updates).

## Files

- `evidence-bundle.schema.json`: The current version (v1) of the schema.
