# Golden Path Onboarding PR Template

Use this template when onboarding a service to the Golden Path. Copy the
contents of `PR_TEMPLATE.md` into your pull request description and fill in the
placeholders before requesting review.

## What's Included

- `PR_TEMPLATE.md`: the PR body template with placeholders for SLOs, alerting,
  runbook references, and disclosure pack config.
- `disclosure-pack.yaml`: a starter configuration file for evidence pack metadata.

## How to Use

1. Copy `PR_TEMPLATE.md` into your PR description.
2. Update the SLO, alerting, and runbook placeholders with service-specific values.
3. Customize `disclosure-pack.yaml` and link it in the PR.
