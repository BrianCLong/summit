# Golden Path Onboarding PR Template

Use the assets in this folder to prepare a Golden Path service onboarding PR.

## Whatâ€™s Included

- `PULL_REQUEST_TEMPLATE.md`: the PR body template with placeholders for SLOs, alerting,
  runbook references, and disclosure pack config.
- `disclosure-pack.yaml`: a starter configuration file for evidence pack metadata.

## How to Use

1. Copy `PULL_REQUEST_TEMPLATE.md` into your PR description.
2. Update the SLO, alerting, and runbook placeholders with service-specific values.
3. Customize `disclosure-pack.yaml` and link it in the PR.
