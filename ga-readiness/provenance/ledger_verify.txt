# Ledger Verification Procedure

Run the verification tool against the exported audit bundle.

```bash
# Verify integrity
./tools/bin/verify-ledger --input ./evidence/audit.log --key ./keys/public.pem

# Expected Output:
# [INFO] Verifying chain integrity...
# [PASS] 15,420 entries verified.
# [PASS] Hash chain intact.
# [PASS] Signatures valid.
# [INFO] Verification complete. Status: VALID
```
