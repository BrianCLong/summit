# Analyst User Guide

**Role:** Intelligence Analyst
**Goal:** Investigate threats using the IntelGraph Knowledge Graph.

## Getting Started
1.  **Login:** Access the portal at `https://graph.intelgraph.io`.
2.  **Dashboard:** View your assigned cases and recent alerts.

## Core Workflows

### 1. Search & Discovery
*   Use the **Global Search** bar to find Entities (People, Orgs, Locations).
*   Use **Filters** to narrow down by Time, Type, or Source.
*   *Tip:* Use boolean operators (AND, OR) for complex queries.

### 2. Graph Exploration
*   Click "Explore" on any entity to view the **Graph Canvas**.
*   **Expand:** Double-click a node to reveal connected entities (1-hop).
*   **Pathfinding:** Select two nodes to find the shortest path or connection.

### 3. Report Generation
*   Add entities to your **Case File**.
*   Click "Generate Brief" to create a PDF report with lineage and provenance data.
