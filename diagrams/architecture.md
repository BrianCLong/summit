# Architecture Overview

## Regulatory Advantage Architecture

*   **Control Plane:** Decides *what is allowed*. Defined in `policy/`.
*   **Application Plane:** Ships features normally.
*   **Data Plane:** Proves everything after the fact. Defined in `compliance/`.
*   **Intelligence Plane:** Watches competitors lock themselves in. Defined in `intelligence/`.

See `diagrams/*.mmd` for Mermaid diagrams.
