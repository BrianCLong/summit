# Security Architecture & Baselines

This directory contains threat models, security policies, and operational runbooks that govern IntelGraph hardening efforts. Documents here are audit-ready and map to platform features, including auth, data governance, and availability.

- `threat-models/`: STRIDE summaries and architecture-specific threat analyses.
- `policies/`: Data retention, access reviews, and related governance controls.
- `../ops/runbooks/`: Operational runbooks for incident response and backup/restore.
