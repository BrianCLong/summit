### Artifact Verification
All container images are signed with **Cosign** (Keyless OIDC) and include **SLSA L3 Provenance** and **SPDX SBOMs**.

**Verify Image Signature:**
```bash
cosign verify \
  --certificate-oidc-issuer https://token.actions.githubusercontent.com \
  --certificate-identity-regexp "^https://github.com/BrianCLong/summit/.+" \
  ghcr.io/brianclong/summit/intelgraph-server:v10.0.0
```

**Verify SBOM & Provenance:**
```bash
# Verify SBOM
cosign verify-attestation --type https://spdx.dev/Document \
  --certificate-oidc-issuer https://token.actions.githubusercontent.com \
  --certificate-identity-regexp "^https://github.com/BrianCLong/summit/.+" \
  ghcr.io/brianclong/summit/intelgraph-server:v10.0.0

# Verify SLSA Provenance
cosign verify-attestation --type https://slsa.dev/provenance/v0.2 \
  --certificate-oidc-issuer https://token.actions.githubusercontent.com \
  --certificate-identity-regexp "^https://github.com/BrianCLong/summit/.+" \
  ghcr.io/brianclong/summit/intelgraph-server:v10.0.0
```
