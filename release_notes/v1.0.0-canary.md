# Release Notes - v1.0.0-canary

## Overview

This is the first canary release of the IntelGraph platform tailored for **Acme Corp**.

## Changes

- **Infrastructure**: Terraform configurations tailored for `us-west-2` and `prod-canary` environment.
- **Helm Charts**: Updated image tags to `v1.0.0` and enabled Ingress for `intelgraph.acme.com`.
- **Governance**: Updated CODEOWNERS to reflect Acme Corp team structure.

## Artifacts

- **Helm Chart**: `charts/intelgraph/`
- **Terraform State**: `s3://summit-tf-state-acme-prod/global/terraform.tfstate`
- **Evidence**: See `evidence/v1.0.0-canary/` for compliance bundles.

## Deployment

Run `bash deploy_canary.sh` to deploy this release.
